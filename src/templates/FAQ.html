Frequently Asked Questions

What is the Ghana Panel Survey?
The Ghana Panel Survey is a longitudinal study that collects data on various socioeconomic indicators from individuals and households in Ghana over multiple waves.
How do I upload data?
Click on the "Choose File" button on the main page, select your CSV file from your computer, and then click "Open".
What file formats are supported?
Currently, only CSV (Comma-Separated Values) files are supported for data upload.
What are the different model types available?
The tool supports several machine learning models: Decision Tree, Random Forest, Gradient Boosting, and Logistic Regression. You can choose one based on your analysis needs.
What is a target variable?
A target variable is the variable that you want to predict using the machine learning model. In this tool, you can select one or more target variables from the dataset.
How do I select multiple target variables?
To select multiple target variables, hold down the "Ctrl" key (or "Cmd" key on Mac) and click on the desired variables.
What happens after I click "Process"?
After clicking "Process", the tool will process the uploaded data, train the selected model(s), and display the feature importances for the chosen target variables.
