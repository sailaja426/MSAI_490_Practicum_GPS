What We Are Doing

The Ghana Panel Survey aims to analyze and track various socioeconomic indicators over time within different demographics. This project utilizes machine learning models to predict outcomes based on the survey data collected.

How to Use This Tool

Follow these steps to use the data upload and model selection tool:

  1. Upload your CSV file: Click on the "Choose File" button to select and upload your dataset in CSV format.
  2. Select Model Type: Choose the type of machine learning model you want to use for analysis. The options include Decision Tree, Random Forest, Gradient Boosting, and Logistic Regression.
  3. Select Target Variable: Select one or more target variables that you want to predict using the chosen model. You can select multiple variables by holding the "Ctrl" key (or "Cmd" key on Mac) while clicking.
  4. Process: Click the "Process" button to start the analysis. The tool will process the data, train the model, and display the feature importances.
